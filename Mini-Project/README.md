# ECommerceApp - Підсумковий міні-проєкт

**Студент:** Gupaliuk Roman
**Варіант:** 2 (Інтернет-магазин)

## Опис
Система управління інтернет-магазином, розроблена як підсумковий проєкт курсу ООП. 
Проєкт демонструє використання Clean Architecture, SOLID, Design Patterns (Strategy, Repository, Facade) та Unit/Integration тестування.

## Структура
* `Domain` - Ядро системи (сутності, бізнес-правила).
* `Application` - Use cases (сервіси, інтерфейси репозиторіїв).
* `Infrastructure` - Робота з файловою системою (збереження стану).
* `Console` - UI шар (взаємодія з користувачем).
