# USER GUIDE

Коротко про використання застосунку ECommerceApp

## Мета
Консольний застосунок для демонстрації базових e-commerce сценаріїв: перегляд/додавання товарів, створення замовлень, збереження та відновлення даних з JSON.

## Швидкий старт (локально)
1. Відкрити рішення в Visual Studio або через `dotnet` CLI.
2. Перейти в проєкт `ECommerceApp.Console` і запустити:

```powershell
dotnet run --project src/ECommerceApp.Console/ECommerceApp.Console.csproj
```

3. Слідувати підказкам у консолі (додавання продуктів, створення замовлення).

## Демонстраційні сценарії
- Створити продукт і зберегти в репозиторій.
- Додати продукт у замовлення і виконати оплату (позитивний сценарій).
- Спроба оплатити порожнє замовлення (негативний сценарій).

## Файли конфігурації та дані
- Файловий репозиторій продуктів: `Infrastructure/Persistence/*.json` (шлях задається в конфігурації або в тестах).

## Відомі обмеження
- Немає GUI; взаємодія через консоль.
- Файлова persistence не оптимізована для великого навантаження.
