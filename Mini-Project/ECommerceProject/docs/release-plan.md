# Release Plan — v1.0.0

Короткий план релізу для версії v1.0.0.

## Що входить у версію v1.0.0
- Стабільні use-cases для створення замовлення, управління продуктами і збереження/завантаження через JSON repository.
- Юніт- та інтеграційні тести (набір у `tests/ECommerceApp.Tests`).
- Базова CLI/Console-демонстрація через `ECommerceApp.Console`.

## Що переноситься в «після курсу» (out-of-scope для v1.0.0)
- Масштабна оптимізація продуктивності для великих наборів даних (bulk async import/export).
- Додаткові патерни (Observer, Proxy, Decorator) як окремі модулі.
- Розгорнутий GUI або веб-інтерфейс.

## Допустимий технічний борг
- Відсутність кешування при доступі до файлів JSON.
- Обмежена валідація вхідних даних на UI рівні (Console app).

## Теми курсу — статус покриття
- SOLID: використано в основних сервісах та інтерфейсах — Частково/Повністю.
- Тести і CI: налаштовано базові юніт/інтеграційні тести — Повністю.
- LINQ/колекції: використання у підрахунках і агрегаціях — Повністю.

## Ризики/завдання перед тегом
- Перевірити останні failing warnings у CI (якщо є).
- Згенерувати CHANGELOG і FINAL_REPORT.
