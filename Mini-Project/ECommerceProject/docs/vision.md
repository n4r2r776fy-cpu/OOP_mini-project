# Project Vision: Інтернет-магазин

## Мета
Створити консольну систему для управління замовленнями інтернет-магазину.

## Бізнес-логіка (не просто CRUD)
Система повинна не лише зберігати дані, але й керувати бізнес-процесами:
1. Валідація залишків на складі при додаванні товару в кошик.
2. Динамічний розрахунок знижок (наприклад, знижка постійного клієнта або святкова знижка).
3. Зміна статусів замовлення (New -> Paid -> Shipped).

## Патерни проєктування
* **Strategy:** Для розрахунку різних типів знижок.
* **Repository:** Для абстрагування логіки збереження даних у файл.
* **Facade:** Для спрощення процесу оформлення замовлення (Check-out).
