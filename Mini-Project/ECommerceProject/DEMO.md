# DEMO — 3–5 хвилинний сценарій

1. Пуск: запустити консольний застосунок

```powershell
dotnet run --project src/ECommerceApp.Console/ECommerceApp.Console.csproj
```

2. Ключовий сценарій (1.5–2 хв):
- Додати 2 продукти (A, B) через консоль.
- Створити замовлення, додати позиції A і B.
- Виконати оплату — показати зміну статусу замовлення.
- Показати, що `TotalAmount` підраховано вірно.

3. Негативний сценарій (30–45 с):
- Спроба оплатити порожнє замовлення — показати викидання помилки/повідомлення.

4. Збереження/відновлення (30–45 с):
- Зберегти продукти в JSON через репозиторій.
- Перезапустити застосунок і показати, що продукти відновились після `LoadAsync`.

5. Пояснення архітектури (30 с):
- Чому Domain/Service/Infrastructure розділені; роль `JsonProductRepository`.

Примітка: додати 5 слайдів презентації (описано нижче).
